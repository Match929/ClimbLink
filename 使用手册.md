## 欢迎使用ClimbLink

ClimbLink是一个专为攀岩爱好者设计的社交平台，让你轻松找到攀岩伙伴、发现优质岩馆、分享攀岩体验。

### 核心特色

- 🤖 **AI智能匹配** - 智能推荐最佳攀岩伙伴
- 🎁 **神秘盲盒** - 随机匹配，惊喜不断
- 🏋️ **岩馆信息** - 全面的岩馆数据，包括设施、价格、活动
- 🎉 **社交互动** - 分享攀岩动态，结识志同道合的朋友
- 🛒 **装备市场** - 二手装备交易平台

### 技术实现状态

以下符号用于标注各功能的实现状态：
- ✅ **数据库实现** - 功能已对接微信云开发数据库，数据持久化存储
- 🔄 **混合实现** - 有数据库API，但包含模拟数据作为fallback
- 🧪 **前端模拟** - 纯前端实现，数据不持久化

## 主要功能介绍

### 1. 首页 (Index)

首页是你探索ClimbLink的起点，包含：

- **盲盒匹配入口** - 快速开始随机匹配 🧪前端模拟
- **热门活动** - 查看最新攀岩活动 🧪前端模拟
- **功能导航** - 快速访问新手指南、装备市场、智能匹配
- **热门岩馆** - 查看附近受欢迎的攀岩馆 🔄混合实现

**实现状态说明：**
- 热门岩馆：使用数据库API，但包含模拟数据作为fallback（如果数据库中没有数据）
- 盲盒和活动：目前为前端模拟数据

### 2. 智能匹配 (Smart Partner)

最强大的功能之一，AI帮你找到最合适的攀岩伙伴：

- **筛选条件**
  - 攀岩水平 (V0-V9+)
  - 时间段 (早/中/晚/自定义)
  - 兴趣标签 (抱石/顶绳/先锋/新手友好等)

- **AI智能评分**
  - 根据你的条件和对方资料综合评分
  - 优先展示匹配度最高的伙伴

- **如何使用**
  1. 打开"Smart Partner"页面
  2. 点击"AI智能匹配"开关
  3. 设置你的筛选条件
  4. 查看匹配结果，选择心仪的伙伴

**实现状态：🧪前端模拟**
- 智能匹配算法为前端实现
- 匹配数据来自前端模拟
- 从岩馆跳转的功能已实现，但数据为模拟

### 3. 盲盒匹配 (Mystery Match)

充满惊喜的随机匹配功能：

- 点击盲盒卡片，系统随机为你匹配一位攀岩伙伴
- 适合喜欢新鲜感的用户
- 每次匹配都是全新体验

**实现状态：🧪前端模拟**
- 匹配逻辑为纯前端随机算法
- 数据不持久化

### 4. 岩馆 (Venues)

全面的岩馆信息平台：

- **地图模式** - 在地图上查看附近岩馆 🧪前端模拟
- **列表模式** - 浏览所有岩馆 🔄混合实现
- **岩馆详情** 🔄混合实现
  - 基本信息 (地址、电话、营业时间)
  - 设施介绍 (抱石区、顶绳区、装备租赁等)
  - 价格信息 (单次票、次卡、月卡)
  - 活动信息 (课程、比赛等)
  - 用户评价

**实现状态：🔄混合实现**
- 有完整的数据库API（getVenues, getVenueById）
- 如果数据库中没有数据，会自动使用模拟数据作为fallback
- 已修复_id和id字段的兼容问题

## 详细使用指南

### 如何寻找攀岩伙伴

#### 方式一：智能匹配 (推荐) 🧪前端模拟

1. 进入"Smart Partner"页面
2. 点击"AI智能匹配"开关
3. 选择你的攀岩水平或点击"All Levels"
4. 选择时间段：
   - "Any Time" - 不限时间
   - "Morning" - 上午 (6:00-12:00)
   - "Afternoon" - 下午 (12:00-18:00)
   - "Evening" - 晚上 (18:00-22:00)
   - "Night" - 夜间 (22:00-6:00)
   - 自定义时间范围
5. 选择兴趣标签 (可多选)
6. 系统会自动计算匹配度并排序
7. 点击感兴趣的伙伴，查看详情或发起聊天

#### 方式二：盲盒匹配 🧪前端模拟

1. 从首页点击"✨ Mystery Match"
2. 进入盲盒页面
3. 点击盲盒，获得随机匹配
4. 开始聊天和约爬

#### 方式三：从岩馆详情页 🔄混合实现

1. 浏览岩馆列表
2. 点击感兴趣的岩馆
3. 点击"Find Partner"按钮
4. 自动跳转到智能匹配页面

### 如何发布约爬请求 ✅数据库实现

1. 进入"Climb Request"页面
2. 填写约爬信息：
   - 时间
   - 地点 (选择岩馆)
   - 难度要求
   - 标签
   - 详细描述
3. 点击发布
4. 等待其他攀岩者回应

### 如何使用岩馆功能 🔄混合实现

#### 浏览岩馆

1. 进入"Venues"页面
2. 可以在地图模式和列表模式之间切换
3. 点击地图标记或列表卡片查看详情

#### 查看岩馆详情

岩馆详情页包含4个标签：

1. **Price** - 价格信息
2. **Facilities** - 设施和装备
3. **Events** - 活动信息
4. **Reviews** - 用户评价

#### 从岩馆找伙伴

1. 在岩馆详情页底部
2. 点击"Find Partner"
3. 自动跳转到智能匹配页面

### 社区功能使用 ✅数据库实现

#### 发布动态

1. 进入"Community"页面
2. 点击发布按钮
3. 添加图片和文字

5. 发布

#### 互动

## 常见问题

### 1. 如何修改个人资料？ ✅数据库实现

进入"Profile"页面，点击编辑按钮即可修改个人信息，数据会自动保存到云数据库。

### 2. 智能匹配的原理是什么？ 🧪前端模拟

AI智能匹配综合考虑：
- 攀岩水平匹配度
- 时间重叠度
- 兴趣标签相似度
- 地理位置接近度

### 3. 盲盒匹配和智能匹配有什么区别？ 🧪前端模拟

- **智能匹配**：根据你的条件筛选，展示匹配度最高的结果
- **盲盒匹配**：完全随机匹配，充满惊喜

### 4. 如何找到特定岩馆的伙伴？ 🔄混合实现

两种方式：
1. 先进入岩馆详情页，点击"Find Partner"
2. 在智能匹配页面，标签筛选相关岩馆

### 5. 如何联系其他用户？ ✅数据库实现

通过消息功能，可以给匹配到的伙伴或社区里的用户发消息，聊天记录会持久化保存。

### 6. 岩馆数据是真实的吗？ 🔄混合实现

目前有完整的数据库API，但也包含模拟数据作为fallback。你可以在数据库中添加真实的岩馆信息，系统会优先使用数据库数据。

### 7. 如何参与活动？ 🧪前端模拟

进入"Events"页面，浏览活动列表，选择感兴趣的活动，查看详情并报名。目前为模拟实现。

### 8. 我可以发布自己的活动吗？ 🧪前端模拟

可以！在活动页面点击创建活动按钮，填写活动信息即可发布。目前为模拟实现。

### 9. 哪些功能的数据会保存？ ✅数据库实现

以下功能的数据会持久化到云数据库：
- 用户注册和个人资料
- 约爬请求
- 社区帖子和评论
- 聊天消息和会话
- 场馆信息（如果已录入数据库）

### 10. 如何启用组件按需注入来优化性能？

建议在项目配置中启用组件按需注入，可以提升小程序启动速度和运行性能，这也是微信官方推荐的最佳实践。

## 技术更新

### 云存储上传功能已实现 ✅

**功能说明：**
- 头像上传：选择图片后自动上传到云存储，保存 fileID 到数据库
- 云存储 API：提供通用文件上传、头像上传、帖子图片上传等功能
- 文件管理：支持删除云文件、获取临时访问链接

**API 说明 (`cloud.js`)：**

```javascript
// 上传头像
const result = await cloud.storage.uploadAvatar(tempFilePath);
// 返回: { success: true, fileID: 'cloud://xxx', cloudPath: 'avatars/xxx' }

// 上传帖子图片
const result = await cloud.storage.uploadPostImage(tempFilePath);

// 通用文件上传
const result = await cloud.storage.uploadFile(tempFilePath, {
  directory: 'uploads',
  fileName: 'custom_name'
});

// 删除文件
await cloud.storage.deleteFile(fileID);

// 获取临时访问链接
await cloud.storage.getTempFileURL(fileID);
```

**使用流程：**
1. 用户选择图片（相册/拍照）
2. 先显示临时图片预览
3. 后台自动上传到云存储
4. 上传成功后替换为 fileID
5. 保存时把 fileID 存入数据库
6. 小程序可直接用 fileID 显示图片
