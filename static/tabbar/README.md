# 图标占位说明

pages.json 中引用的 tabbar 图标需要以下文件：

- static/tabbar/home.png
- static/tabbar/home-active.png
- static/tabbar/community.png
- static/tabbar/community-active.png
- static/tabbar/venues.png
- static/tabbar/venues-active.png
- static/tabbar/profile.png
- static/tabbar/profile-active.png

可以使用图标库（如 iconfont）或在线图标生成工具创建这些图标。
